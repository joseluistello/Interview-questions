### Escribe una funcion que calcule la longitud de una lista

### Solucion 

```py
def longitud_lista(lista):
    contador = 0
    for elemento in lista:
        contador += 1

    return contador

print(longitud_lista(["Pedro", "Armando", "Alfonso", "Jose"]))
```

### Output

```py
4
```
